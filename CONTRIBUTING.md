# Contributing to LLM Provider Pulse

We welcome contributions to make LLM Provider Pulse better!

## How to Contribute

1. Fork the repository
2. Create your feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Adding a New Provider

1. Create a provider config file in `providers/<name>.json`
2. Add the provider info to `scripts/providers.json`
3. Add the provider row to the README comparison table
4. Submit a PR

## Reporting Issues

Please use the issue templates provided in `.github/ISSUE_TEMPLATE/`.
